# Input Images

Place images here for classification.

## Supported Formats
- `.jpg` / `.jpeg`
- `.png`
- `.bmp`
- `.tiff`
- `.webp`

## Usage

1. Copy your guava images to this folder
2. Open `../Guava_Classifier.ipynb`
3. Run the notebook to classify images

## Example

After placing images:
```
input/
├── guava_test_1.jpg
├── guava_test_2.png
└── my_guava.jpeg
```

The classifier will predict the ripeness/age class for each image.
