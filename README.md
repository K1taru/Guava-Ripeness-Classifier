# Guava Ripeness Classifier

Classifies guava ripeness using an Arduino Uno, pH sensor, MQ135 (VOCs/CO2), and MQ3 (ethylene/aromatics) sensors.

Features:
- Multi-sensor ripeness detection logic
- Visual status with LEDs (ripe/unripe)
- Live sensor values and ripeness status on I2C 20x4 LCD

Wiring and calibration required for best results.
